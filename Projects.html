Road Safety using number plate detection

The project helps with enforcing speed limits. The main concept of the project is to have two logging stations which will take a photo of number plate of passing vehicles, this is then feed into the server which will use number plate detection to store the number plate number and timestamp into a database. As the vehicle passes through the second logging station the number plate is again detected, this will then check the backend database for a entry of the vehicle number plate number. If it is found, it will retrieve the timestamp and calculate the average speed of the car using the speed, time, distance formula. If it is found that the average speed is greater than the maximum speed, then the car number is added to a csv file.
